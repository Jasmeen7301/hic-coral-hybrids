# Step 03 — Manual Assembly Curation (Juicebox JBAT)

This step is performed manually using the **Juicebox Assembly Tools (JBAT)** graphical interface and cannot be scripted.

## What to do

1. Download and open [Juicebox](https://github.com/aidenlab/Juicebox)
2. Load the `.hic` file and `.assembly` file generated by YaHS (Step 02)
3. Manually inspect the Hi-C contact map for misjoins — these appear as off-diagonal signal breaks
4. Correct misjoins by clicking and dragging chromosome segments
5. Confirm chromosomal contiguity through long-range interaction patterns
6. Validate structural changes visually
7. Export the curated assembly: **Assembly → Export Assembly**

## Outputs

- `curated_assembly.assembly` — corrected assembly file
- Use this in Step 04 to regenerate Hi-C contact maps with 3D-DNA

## Tips

- Zoom into regions with weak diagonal signal to identify potential misjoins
- Use the 2D contact map to confirm that chromosomes show clean diagonal patterns after curation
- Compare before and after curation by toggling the original and curated `.hic` files
