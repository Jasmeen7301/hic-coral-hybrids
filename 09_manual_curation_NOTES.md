# Step 09 — Manual Assembly Curation (Juicebox JBAT)

This step is performed manually using the Juicebox Assembly Tools (JBAT) graphical interface and cannot be scripted.

## What to do

1. Download and open [Juicebox](https://github.com/aidenlab/Juicebox)
2. Load the `out_JBAT.hic` and `out_JBAT.liftover.assembly` files generated by Steps 07–08
3. Inspect the Hi-C contact map for misjoins — these appear as off-diagonal signal breaks
4. Correct misjoins by clicking and dragging chromosome segments
5. Confirm chromosomal contiguity through long-range interaction patterns
6. Export the curated assembly: **Assembly → Export Assembly**

## Outputs

- `curated.assembly` — corrected assembly file
- Use with 3D-DNA visualisation script (Step 10) to regenerate contact maps

## Tips

- Zoom into regions with weak diagonal signal to identify misjoins
- Sample A was used as the reference genome for all comparative analyses
- Compare before and after curation by toggling the original and curated `.hic` files
